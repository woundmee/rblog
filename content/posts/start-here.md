# Старт блога

Этот сайт построен как **минималистичный блог** с markdown-контентом.

## Что уже есть

- адаптивная главная страница со списком статей
- тёмная и светлая тема по настройкам устройства
- отдельная страница `Обо мне`
- админ-публикация новых статей

## Как писать посты

Можно использовать обычный Markdown:

```ts
const post = {
  title: "Новая статья",
  tags: ["typescript", "nextjs"]
};
```

Пиши коротко, по делу и с примерами.
